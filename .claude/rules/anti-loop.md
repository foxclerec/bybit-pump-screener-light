# Anti-Loop Rules

## Главное правило

Та же ошибка возникла 2 раза после попыток исправления → **СТОП**.
Вывести: текущий код + полный текст ошибки + спросить пользователя что делать.

## Запрещено

- Удалять код чтобы "посмотреть исчезнет ли ошибка"
- Переписывать целые файлы из-за небольшого бага
- Менять сигнатуры функций без предварительного grep всех мест вызова

## Атомарные изменения

Один шаг = одно изменение в одном модуле, затем проверка.
Никогда не менять несколько модулей сразу без промежуточной проверки.

## Высокорисковые файлы

Перед правкой этих файлов — предложить план и дождаться ОК:
- `app/screener/runner.py` — главный цикл скринера
- `app/screener/detectors/pump_rules.py` — логика детекции
- `app/exchanges/bybit/client.py` — API-клиент биржи
- `app/models.py` — ORM-модели базы данных
